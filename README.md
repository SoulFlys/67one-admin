# 67one-admin

> 67one-blog的后台管理系统

## 前置
* 67-api https://github.com/SoulFlys/67-api

## Build Setup

``` shell
# 安装依赖
$ npm install

# 或者
$ yarn

# 开发模式
$ npm run dev

# 生产模式
$ npm run build
```
